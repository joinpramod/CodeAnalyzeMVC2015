Download the .net library and public private key from the link below

 

http://code.google.com/p/recaptcha/downloads/list?q=label:aspnetlib-Latest

 

After Download add reference to recaptcha.dll in the project in visual studio, 

Add the below 2 config values to web.config

 

ReCaptchaPrivateKey

ReCaptchaPublicKey

 

In the view add the below code where you want recaptcha to be displayed

@Html.Raw(Html.GenerateCaptcha())
