1. -> We will see how we can implement Amazon style image zoom where we have multiple images listed one after and mouseover we see a zoomed image on the right.


    -> For each image we will need a magnified image of the same with more clarity. We will be showing the magnified image on mouseover.


    -> We will have to refer the jquery and jquery elevatedzoom image zoom plugin. We can refer this script directly from github using the below link http://cdn.rawgit.com/elevateweb/elevatezoom/master/jquery.elevateZoom-3.0.8.min.js


    -> Consider we have a gridview or a normal html table with an id where in each we have some image. Whole view is as shown below. If you notice for each image we use a property called data-zoom-image


    View 
    
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
<script type="text/javascript" src="http://cdn.rawgit.com/elevateweb/elevatezoom/master/jquery.elevateZoom-3.0.8.min.js"></script>
<script type="text/javascript">
    $(function () {
        $("[id*=myTable] img").elevateZoom({
            cursor: 'pointer',
            imageCrossfade: true,
            loadingIcon: 'loading.gif'
        });
    });
</script>

<table cellspacing="0" rules="all" border="1" id="myTable" style="border-collapse:collapse;">
		<tr>
			<td>
                    <img src='/Images/Small/SmallApple.jpg' alt="" data-zoom-image='/Images/Large/LargeApple.jpg' />
                </td>
		</tr><tr>
			<td>
                    <img src='/Images/Small/SmallOrange.jpg' alt="" data-zoom-image='/Images/Large/LargeOrange.jpg' />
                </td>
