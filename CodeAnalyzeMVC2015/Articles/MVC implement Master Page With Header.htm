Layout.cshtml inside Views -> Shared folder acts as Master Page in ASP.NET MVC application

 

So to have Header and Footer we need to add to more partial views as shown below

 

 

 

Code is given below

 

 

<body>
    <header>
        <div class="content-wrapper">
            @Html.Partial("_Header")
        </div>
    </header>

    <div id="body">
        <section class="content-wrapper main-content clear-fix">
            @RenderBody()
        </section>
    </div>
    <footer>

        <div class="content-wrapper">
            <div class="float-left">
                @Html.Partial("_Footer")
             </div>
        </div>
    </footer>

    @Scripts.Render("~/bundles/jquery")
    @RenderSection("scripts", required: false)
</body>

 

 

 

While adding a view select the Layout.cshtml as Master Page for your new view.
