1. Asp.Net MVC have an in built chart helper class using which we can draw various types of charts like pie, line, bar, stock, area, column charts.
We will have input the data for the chart to be drawn. This can be either static or dynamic. The data can be an array or from an xml file or data can be got from database.
Take a look at the demo to get better understanding.
  2. View Demo

  3. View code 
    //Bar chart
var myChart1 = new Chart(width: 400, height: 300)
        .AddTitle("Bar Chart")
        .AddSeries(
            name: "Bar", chartType: "Bar",
            xValue: new[] {  "Emp1", "Emp2", "Emp3", "Emp4", "Emp5" },
            yValues: new[] { "2", "6", "4", "5", "3" })
        .Write();

//Pie chart
var myChart2 = new Chart(width: 400, height: 300)
            .AddTitle("Pie Chart")

                xValue: new[] {  "Emp6", "Emp7", "Emp8", "Emp9", "Emp10" },
