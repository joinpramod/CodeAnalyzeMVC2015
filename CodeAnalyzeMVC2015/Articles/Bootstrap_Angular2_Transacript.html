Bootstrap angular 2 application having component with transcript

  1. Lets take a look at bootstraping angular 2 application with transscript
    We will need the following import in our application after having angular 2 installed.

    <script src="//code.angularjs.org/2.0.0-beta.8/angular2-polyfills.js"></script>
<script src="//code.angularjs.org/tools/system.js"></script>
<script src="//code.angularjs.org/tools/typescript.js"></script>
<script src="//code.angularjs.org/2.0.0-beta.8/Rx.js"></script>
<script src="//code.angularjs.org/2.0.0-beta.8/angular2.min.js"></script>

  2. Here is the html file or you can say the base before creating the app having all the major angular 2 includes. We will be using trascript here 

    <!DOCTYPE html>
<html>
  <head>
    <title>Angular 2</title>
    <link rel="stylesheet" href="style.css" />
    <script src="//code.angularjs.org/2.0.0-beta.8/angular2-polyfills.js"></script>
    <script src="//code.angularjs.org/tools/system.js"></script>
    <script src="//code.angularjs.org/tools/typescript.js"></script>
    <script src="//code.angularjs.org/2.0.0-beta.8/Rx.js"></script>
    <script src="//code.angularjs.org/2.0.0-beta.8/angular2.min.js"></script>
    <script>
        System.config({
            transpiler: 'typescript',
            typescriptOptions: {
                emitDecoratorMetadata: true
            },
            map: {
                app: './app'
            },
            packages: {
                app: {
                    main: './main.ts',
                    defaultExtension: 'ts'
                }
            }
        });

        System
          .import('app')
          .catch(console.error.bind(console));
    </script>
  </head>
  <body>
    <my-app>
      Loading...
    </my-app>
  </body>
</html>

  3. Next we need to create component as angular 2 is nothing but component Here is the component of the angular 2 i.e. app.component.ts This has a simple message. 

    // app.component.ts
import {Component} from 'angular2/core';

@Component({
  selector: 'my-app',
  template: `
    
    
      Hi! How are you?
    
    
  `
})
export class AppComponent {
  
}

  4. Code to bootstrap the app main.ts. We will be bootstraping from within angular 2 i.e. from angular2/platform/browser and this should be done. 

    // main.ts
import {bootstrap} from 'angular2/platform/browser';
import {AppComponent} from './app.component';

bootstrap(AppComponent);
