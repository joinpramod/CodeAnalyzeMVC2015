Here is the procedure to use areas in asp.net core mvc.
1. We need to edit and update configure method in startup.cs file as shown below.
2. Create a folder named Areas in the root folder and within this create the necessary MVC folders like View, Controllers and so on.
3. Lets create couple of controllers called EmployeeController and ManagerController.
4. Look at the comments for more information on the code

<!--startup.cs-->
<!--Make the changes for startup.cs file as shown below-->
app.UseMvc(routes =>
{
    routes.MapRoute("areaRoute", "{area:exists}/{controller=Employee}/{action=Index}/{id?}");

    routes.MapRoute(
        name: "default",
        template: "{controller=Home}/{action=Index}/{id?}");
});



<!--TestController.cs-->
<!--TestController inside Controllers folder in Areas-->
<!--Make sure to create required Views in Views folders for all the actions in Controller-->
<!--Also the route here is also important-->
