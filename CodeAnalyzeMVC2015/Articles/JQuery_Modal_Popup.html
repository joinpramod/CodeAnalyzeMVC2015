1. This articles explains about displaying or showing modal popup on button click or link click or any action from user in asp.net mvc or any other web application.
We wil display the contents in div on a modal popup. Jquery modal popup has following important properties

         modal         	--> boolean value to indicate if the control it is being tied is modal or not, value is true for this
         autoOpen   	--> by default autoOpen is always true which mean this will popup when the page loads
         title          	--> title of popup window
         width         	--> width of modal popup                 
         height        	--> heigh of modal popup

We need to have mandatory reference to the following scripts and style sheets
        jquery.min.js
        jquery-ui.js    --> can be donwloaded from https://jqueryui.com/download/

        jquery-ui.css   --> can be donwloaded from https://jqueryui.com/download/

  3. Sample code 
    <script src="../Scripts/jquery-1.10.2.min.js"></script>
 <script type="text/javascript">
     $(function () {
         $("#dialog").dialog({
             modal: true,
             autoOpen: false,
             title: "jQuery modal popup example",
             width: 300,
             height: 150
         });
         $("#btnShow").click(function () {
             $('#dialog').dialog('open');
         });
     });
</script>
<input type="button" id="btnShow" value="Show Popup" />
<div id="dialog" style="display:none">
Hi, how are you?
</div>
