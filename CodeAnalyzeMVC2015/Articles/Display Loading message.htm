This can be achieved in jquery

 

Add below code in View

<script src="~/Scripts/jquery-1.8.2.min.js"></script>

<script src="@Url.Content("~/Scripts/jquery.unobtrusive-ajax.min.js")" type="text/javascript"></script>

 

Place the view part which is supposed to be processed with ajax in ajax form as shown below

 

  @using (Ajax.BeginForm(new AjaxOptions { HttpMethod = "POST", LoadingElementId = "loading1" }))

    {               

        <div id="loading1" style="display: none;">

            <div class="modal"></div>

            <div class="spinner">Loading...</div>

        </div>

        <input type="submit" value="Submit" />

    }

 

 

 

 

Use the below css class to display spinner and message

<style>

            #please-wait {

                position: absolute;

                left: 0;

                top: 0;

                height: 100%;

            }

 

                #please-wait .modal {

                    z-index: 1999;

                    left: 0;

                    top: 0;

                    width: 100%;

                    height: 100%;

                    opacity: 0.5;

                    -moz-opacity: 0.5;

                    background-color: black;

                    margin-left: 0;

                }

 

                #please-wait .spinner {

                    z-index: 2000;

                    position: absolute;

                    font-weight: bold;

                    text-align: center;

                }

     </style>

 

Sample controller

 

        [HttpPost]

        public ActionResult Index()

        {

            Thread.Sleep(3000);

            return View();

        }

 

If needed add below config value in web.config

 

    <add key="webpages:Enabled" value="false" />

    <add key="ClientValidationEnabled" value="true" />

    <add key="UnobtrusiveJavaScriptEnabled" value="true" />
