1. This article explains about implementing javascript richtext editor control in asp.net mvc.

We will be using TinyMCE for this purpose, over the time TinyMCE has become one of the most popular and reputed editor control
for most of the web application. This is because it is customizable and uses pure javascript. Download the whole TinyMCE Dev Package from here
https://www.tinymce.com/download/. 

Unzip and copy the whole tinymce folder which contains all javascript and jar files into your project folder which you can refer from application.

You can also at the basic and full version Demos of this here https://www.tinymce.com/docs/demo/basic-example/ with which you will get and idea
that how simple this is to implement and use

After this we will need to integrate this tinymce into our MVC application. We will refer the tinymce javascripts first like this,
    View 
    <script src="~/Scripts/tinymce/tinymce.min.js"></script>
  2. View Demo of Code

  3. We will need a TextArea mvc control, TinyMce will look for TextArea control and convert it into full rich text editor Editor
        
    @Html.TextArea("txtMVCEditor")
        
  4. Whole View will look some thing like           
            
    <script type="text/javascript" src="~/Scripts/tinymce/tinymce.min.js"></script>
<script type="text/javascript">
    tinymce.init(
        {
            mode: "textareas",
            statusbar: false,
            height: "300px",
            width: "98%"
    });
</script>
@using (Html.BeginForm("ProcessEditor", "MyController", FormMethod.Post))
{
<table style="width:100%; font-size:16px">
    <tr>
        <td>
            <table>               
                <tr>
                    <td align="left">
                        @Html.TextArea("MyMVCEditor")
                    </td>

                <tr>

                        <input type="submit" value="Submit" id="btnSubmit" />
                    </td>
                </tr>

        </td>
    </tr>
</table>
