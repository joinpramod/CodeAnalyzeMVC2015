Create tabular layout in objective-C for iOS app using XCode to show images Create grid layout in objective-C for iPhone app using XCode to show images

  1. We will use UICollectionView to create grid like structure or table like structure in iOS app.

  2. Create a XCode project and name it as per your convenience.

  3. In Main.Storyboard, delete the default view controller and add a new CollectionView controller from the object library.
    Change the size of the cell if needed by selecting the cell that appears ofter adding the view.

  4. Insert image inside the cell by adding image view controller and label it if needed.

  5. Create a new class inherited from UICollectionViewCell to customize the cell as needed. Lets call it as SubCell. Change the class of the cell to this new class SubCell and set reuseidentifier as cell.

  6. Create another new class inherited from UICollectionViewController. Lets call it CustomViewController. Select view controller in the storyboard and change its class from default to this CustomViewController. Delete existing .m and .h files.

  7. Implement methods like UICollectionViewDataSource in CustomViewController.h file to use UICollectionView methods. Create an mutable array to store objects for cell and a mutable dictionary for each object in CustomViewController.h file 

    #import  <UIKit/UIKit.h>

@interface CustomViewController : UICollectionViewController <UICollectionViewDataSource>

@property (nonatomic,strong) NSMutableArray *arrImages;
@property (nonatomic,strong) NSMutableDictionary *dictImages;

@end

  8. SubCell.h file code will look like below

    @interface SubCell : UICollectionViewCell
@property (weak, nonatomic) IBOutletUIImageView *ivImage;
@property (weak, nonatomic) IBOutletUILabel *lblPosition;
@end

  9. Also implement NumberOfItemsInSection, cellForItemAtIndexPath in CustomViewController.m file. 

    - (NSInteger)collectionView:(UICollectionView *)collectionView numberOfItemsInSection:(NSInteger)section {
    return [arrImages count];
}

- (UICollectionViewCell *)collectionView:(UICollectionView *)collectionView cellForItemAtIndexPath:(NSIndexPath *)indexPath
{
    SubCell *cell = [collectionView dequeueReusableCellWithReuseIdentifier:reuseIdentifier forIndexPath:indexPath];    
    cell.lblIndex.text=[[arrImages objectAtIndex:[indexPath row]]valueForKey:@"Info"];    
    UIImage *image=[UIImage imageNamed:[[arrImages objectAtIndex:[indexPath row]]valueForKey:@"imageFile"]];    
    [cell.ivCartoon setImage:image];    
    return cell;
}
