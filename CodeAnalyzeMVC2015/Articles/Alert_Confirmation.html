1. This article explains about creating alert confirmation window and then processing the user selected option of OK or Cancel with Yes and No on the server side.
Normally while performing delete actions we would need to double confirm the user action of deletion as there might a chance of user clicking delete by mistake or not intended to do it.
With this option we can make the user sure of the action.
We will show the javascript alert confimration window and we pass the user selected option via a hidden filed to the server by post method.
We validate this value on the server side and process accordingly. Take a look at the sample.
  2. View Demo

  3. View Code 
    @using (Html.BeginForm("AlertConfirmation", "Code", FormMethod.Post))
{
    <input type="submit" value="Show alert confirmation box" onclick="Confirm()"  style="font-weight: bold; font-family: Calibri; color: green; font-size:20px"/>
    <input type="hidden" id="hdnValue" name="hiddenValue" />
}

<br /><span style="width: 100%; font-weight: bold; font-family: Calibri; color: Red; font-size:20px">@ViewBag.Message</span>



<script type="text/javascript">
    function Confirm() {
        if (confirm("Get this confirmation from server")) {
            document.getElementById("hdnValue").value = "Yes";
        } else {
            document.getElementById("hdnValue").value = "No";
        }
    }
</script>
