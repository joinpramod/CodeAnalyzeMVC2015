This article explains about implementing treeview using css and jquery.

Treeview can quite challenging at times especially in web based applications, because binding the data and get the animation takes lots of our time unless we dont have any third party
libraries dlls like telerik or infragistics etc. 

Here is a sample implementation of tree using only css and jquery which is not that hard to implement and one should understand by looking at the code as it is self explanatory.

Basically we have this method called LoadChildren which will display all the child nodes based on Node selection, so for this we will need another method called NodeSelection to highlight
the particular node the user selects. Take a look at the demo to see this in action and to get some idea.

  6. Javascript 
    <script src="~/Scripts/jquery-1.10.2.min.js"></script>        

<script type="text/javascript">
    function LoadChildren(ctrl) {
        if ($(ctrl).next('a').next('ul').is(':visible')) {
            $(ctrl).next('a').next('ul').hide('slow');
            $(ctrl).attr('src', 'blueright.png');
            $(ctrl).attr('style', 'width:20px;height:20px');
        }
        else {
            $(ctrl).next('a').next('ul').show('slow');
            $(ctrl).attr('src', 'blueright.png');
            $(ctrl).attr('style', 'width:20px;height:20px');
        }
    }


    function NodeSelection(ctrl) {
        $('ul.tree li').find('a').each(function () { $(this).removeClass(); })
        $(ctrl).addClass("selected");
    }

    $(document).ready(function () {
        $('ul.tree li').each(function () {
            $(this).children('img').click(function () {
                LoadChildren(this);
            });
        });

        $('ul.tree li a').click(function (e) {
            $('ul.tree li').find('a').each(function () { $(this).removeClass(); })

            $(this).addClass("selected");
        });
    });
</script>
