1. In this article we will see how to validate email address at the client side using jquery in asp.net mvc. To do this we will need jquery validation script jqueryval.js which we will be refering at the client side.
    This script jqueryval.js will handle the validation part at the client side.
    The Required Data Annotation and the EmailAddress Data Annotation attributes have been specified with a property Error Message with a string value.


    Model 
                    
public class Employee
{
    [Display(Name = "Email Address")]
    [Required(ErrorMessage = "Email is required.")]
    [EmailAddress(ErrorMessage = "Invalid Email Address.")]
    public string Email { get; set; }
}

  2. Controller will be same with no special changes and with a Post method. Take a look at the View code below


    3. View 
             
<body>
    @using (Html.BeginForm("Index", "Home", FormMethod.Post))
    {
        <table>
            <tr>
                <td>@Html.DisplayFor(m => m.Email)</td>
                <td>@Html.TextBoxFor(m => m.Email)</td>
                <td>@Html.ValidationMessageFor(m => m.Email, "")</td>
            </tr>
            <tr>
                <td></td>
                <td><input type="submit" value="Submit"/></td>
                <td></td>
            </tr>
        </table>
    }
</body>
@Scripts.Render("~/bundles/jquery")
@Scripts.Render("~/bundles/jqueryval")
</html>
