Pass Data to Partial View using ViewBag/ViewData

       public ActionResult Index()

        {

            ViewBag.Name = "CodeAnalyze";

            return View();

        }          

 

       @Html.Partial("MyPartialView", (string) @ViewBag.Name)

 

In controller this can be got by @Model variable

 

Pass Data to Partial View using TempData

       public ActionResult Index()

        {

            TempData["Name"] = "CodeAnalyze";

            return View();

        }          

 

       @Html.Partial("MyPartialView", (double) TempData["name"])

 

In controller this can be got by @Model variable

 

 

Passing from View to Partial View

                @{

              ViewBag.Title = "Index";

              string name = "CodeAnalyze";

       }

 

       <h2>Index</h2>

       @Html.Partial("MyPartialView", name)

In PartialView this value can be got from @Model variable
