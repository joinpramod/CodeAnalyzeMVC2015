1. Download jquery plugin from https://plugins.jquery.com/datetimepicker/
2. Refer jquery.datetimepicker.min.js script in the View
3. Also refer datetimepicker.css file in the view
4. We can set the current date and time before loaidng the control which is given below
5. Take a look athe demo and code Below is the code of the view to select date and time using datetimepicker juqry plugin in asp.net mvc

View Demo

    <link href="../jquery.datetimepicker.css" rel="stylesheet" />
    <script src="../jquery.datetimepicker.min.js"></script>
     <script src="../jquery.js"></script>

    <!--View-->
    <script type="text/javascript">
        jQuery('#datetimepicker').datetimepicker();
    </script>

    <div class="container">
    <div class="row">
       @Html.Label("DateTime using Jquery plug in",null, new { @class = "control-label col-sm-3" })
               <div class="col-sm-3"> 
                @Html.TextBox("datetimepicker")
               </div>
     </div>  
    </div>
